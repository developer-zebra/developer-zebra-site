## Download EHS
Using the latest release is recommended for most cases. EHS 2.5 includes a single .apk for installing to all [devices supported by v2.5](../guide/about#supporteddevices). Download packages prior to EHS 2.5 include separate .apk packages for Jelly Bean and KitKat, plus an optional installer .apk that detects the target operating system and installs the correct EHS version automatically.

After download, proceed to the [EHS 2.5 Setup page](/ehs/2-5/guide/setup). 

## Latest Release

Visit the [Enterprise Home Screen download page](https://www.zebra.com/us/en/support-downloads/software/utilities/enterprise-home-screen.html) to download Enterprise Home Screen 2.5.
